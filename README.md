# 💰 Expense Tracker – Desktop Application

A multi-user **Desktop Expense Tracker Application** built using **Java**, **JavaFX**, **Spring Framework (Basics)**, and **MySQL**.

This application helps users efficiently manage their income and expenses while visualizing financial data through interactive charts and trend analysis.

---

## 🚀 Tech Stack

- **Language:** Java  
- **Frontend:** JavaFX  
- **Backend:** Spring Framework (Basic Configuration)  
- **Database:** MySQL  
- **Architecture:** Desktop-based MVC Pattern  

---

## 🖥️ Application Type

- Desktop Application  
- Multi-user Support  
- Database-driven  
- Real-time Financial Tracking  

---

## ✨ Features

### 🔐 User Management
- Multi-user login system  
- Secure authentication  
- Isolated financial records per user  

### 💵 Income & Expense Management
- Add income entries  
- Add expense entries  
- Categorize expenses (Food, Rent, Travel, etc.)  
- Edit and delete transactions  

### 📊 Data Visualization

#### 🥧 Pie Chart View
- Category-wise expense distribution  
- Identify highest spending areas easily  

#### 📈 Trend Analysis
- Income trend visualization  
- Expense trend visualization  
- Monitor financial growth or decline over time  

### 📋 Financial Insights
- Monthly summary  
- Category-based expense breakdown  
- Clear graphical representation using JavaFX charts  

---

## 🏗️ Project Architecture

Frontend (JavaFX UI)  
↓  
Spring Backend Layer  
↓  
MySQL Database  

- JavaFX handles the user interface and chart visualization  
- Spring manages business logic and application flow  
- MySQL stores user and transaction data  

---

## 🗄️ Database Design

Main Tables:

- Users  
- Income  
- Expenses  
- Categories  

Relational mapping ensures each user’s data is securely separated.
